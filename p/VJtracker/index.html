Tần suất theo dõi giá vé máy bay Vietjet là bao lâu?

Dữ liệu được cập nhật tự động 3 lần mỗi ngày vào lúc 8:00 Sáng, 2:00 Chiều và 8:00 Tối giờ địa phương sử dụng Windows Task Scheduler (hoặc thông qua GitHub Actions trên kho lưu trữ đám mây).

Cách tính gợi ý săn vé máy bay giá rẻ (MUA NGAY vs CHỜ THÊM) hoạt động thế nào?

Gợi ý được tính toán bằng cách so sánh giá quét mới nhất với mức trung bình lịch sử của chặng bay và khoảng thời gian mua trước đó:

  • MUA NGAY (Xu hướng Xanh): Giá hiện tại thấp hơn giá trung bình lịch sử ít nhất 3%.
  • CHỜ THÊM (Xu hướng Đỏ): Giá hiện tại cao hơn giá trung bình lịch sử ít nhất 3%.
  • MUA NGAY (Giá ổn định): Giá hiện tại nằm trong khoảng 3% so với giá trung bình, cho thấy điều kiện đặt vé ổn định.
Làm thế nào để đồng bộ cơ sở dữ liệu cục bộ với GitHub Pages?

Nếu bạn đang chạy công cụ quét cục bộ, bạn có thể dễ dàng đồng bộ cơ sở dữ liệu và cập nhật giao diện trực tuyến bằng cách đẩy tệp JSON mới lên GitHub:

git add flights_db.json
git commit -m "Cập nhật cơ sở dữ liệu giá vé"
git push origin main

Việc này sẽ tự động kích hoạt tiến trình triển khai GitHub Pages để cập nhật trang web hiển thị của bạn.
